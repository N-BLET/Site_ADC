# Site_ADC

### SUJET :

Vous réaliserez un site complet développé en HTML/CSS/JS et en PHP et s’articulant autour d’une base de données MySQL.
Vous devrez utiliser le CSS pour réaliser votre mise en page et du Javascript (et/ou jQuery) pour les traitements côtés client.
Pour les traitements côtés serveur, vous devrez utiliser PHP et une base de données MySQL. L’accès aux données devra être effectuée exclusivement par PDO afin de rendre le site plus portable et sécurisé.
Le but de cette unité étant de développer un site de A à Z, l’utilisation de CMS ou de frameworks PHP n’est donc pas autorisée.
Le thème du site web est laissé à votre convenance, mais les points suivants devront au minimum être traités :
- Système d’inscription/authentification,
- Gestion des sessions et cookies,
- Envoi de mail,
- Vérification des données,
- Gestion d’au moins deux profils avec des droits différents (exemple administrateur et client),
- Utilisation d’une base de données comportant au moins cinq tables,
- Utilisation de requêtes variées (lecture, insertion, modification, suppression) et de formulaires.
